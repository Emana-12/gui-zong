# Game Concept: 归宗 (Gui Zong)

*Created: 2026-04-20*
*Status: Draft*

---

## Elevator Pitch

> 一把剑，一条命，一路归宗。这是一款中国剑侠 3D 动作 Roguelite，玩家通过精确掌握三式剑招（游剑式、钻剑式、绕剑式）的时机与空间，在水墨山水般的竞技场中一路斩敌。每局结束后所有剑魂散去，但你的肌肉记忆留了下来——真正的成长不在属性面板上，在你手中。

## Core Fantasy

**从学徒到剑圣——每一次失败都让你更强，不是数值上的，而是操作上的。**

玩家的幻想不是"我收集了最强装备"，而是"我已经不需要装备了"。当玩家能够用最简练的动作完成最复杂的连击，当万剑归宗因为纯粹的技巧而非运气触发时——那就是剑圣的时刻。

你能做到这里的事在其他任何地方都做不到：用三种截然不同的剑式在瞬间切换中编织战斗——墨色流光护身化解攻击，金色剑气缠绕敌刃触发材质反应，高密度粒子凝聚穿透一切防御——最终在连击巅峰触发万剑归宗。这不是按钮乱按的爽感，是**精确到毫秒的优雅**。

---

## Unique Hook

**Like Hades AND ALSO 你的战斗技巧才是永久进度。**

Roguelike 的成长不是属性提升，是玩家手法的进化。没有"刷属性通关"的退路——每一次进步都必须是真实的操作提升。三式剑招随时独立可用（三键独立），形成一个无限深的战术空间：绕剑式护身化解（以柔克刚），游剑式缠绕敌刃（以巧制敌，材质交互），钻剑式穿透破坏（以力破防）。

## Player Experience Analysis (MDA Framework)

### Target Aesthetics (What the player FEELS)

| Aesthetic | Priority | How We Deliver It |
| ---- | ---- | ---- |
| **Sensation** (感官愉悦) | 1 (Primary) | 墨色流光与金色剑气的视觉优雅 + 精确命中时的音效反馈 + 材质碰撞反应（火花/裂纹/墨点炸碎）+ 剑招节奏感 |
| **Challenge** (挑战与精通) | 2 | 三式组合的无限深度 + 纯技巧无数值成长 + 连击触发万剑归宗 |
| **Fantasy** (剑圣幻想) | 3 | 水墨山水为场景的中国剑侠世界 + 万剑归宗的终极场景 |
| **Discovery** (招式发现) | 4 | 每式 2 个变体 + 三式组合产生的隐藏招式 |
| **Expression** | N/A | 无——不是表达型游戏 |
| **Narrative** | N/A | 无强制叙事（反支柱） |
| **Fellowship** | N/A | 单人（反支柱） |
| **Submission** | N/A | 硬核操作门槛（不适合休闲） |

### Key Dynamics (Emergent behaviors)
- 玩家会在三种剑式之间自然形成个人风格偏好，但最优解要求三式精通——绕剑式化解、游剑式缠绕、钻剑式破防，缺一不可
- 材质交互系统（金属冒火花、木杖裂细纹、墨点炸碎）让每次命中都有独特反馈
- 连击系统的存在促使玩家追求更长、更精准的战斗节奏
- "无伤通关"和"完美连击"会成为玩家自我挑战的目标

### Core Mechanics

1. **三式剑招系统** — 游剑式（金色剑气缠绕敌刃，材质交互）、钻剑式（高密度金色粒子凝聚，穿透破坏）、绕剑式（墨色流光护身，以柔克刚），三键独立随时可用
2. **流光轨迹系统** — 绕剑式产生墨色流光，游剑式和钻剑式产生金色剑气。精确反映剑尖运动，材质碰撞产生对应反应（火花/裂纹/墨点炸碎）
3. **连击/万剑归宗系统** — 三式连击积累，达到阈值触发万剑归宗终极场景
4. **竞技场波次系统** — 随机敌人波次，难度递增，死亡重来
5. **材质交互系统** — 金属冒火花、木杖裂细纹、墨点炸碎，每种材质有独特反馈
6. **纯技巧进度系统** — 无永久属性升级，所有成长来自玩家手法

## Player Motivation Profile

### Primary Psychological Needs Served

| Need | How This Game Satisfies It | Strength |
| ---- | ---- | ---- |
| **Autonomy** (自由选择) | 三式随时可用，打法完全由你决定——先游后钻，还是钻后绕？每场战斗都是即时战术决策 | Core |
| **Competence** (精通成长) | 每次死亡都教会你什么，每次运行都比上次更好。连击长度和万剑归宗频率是可量化的进步指标 | Core |
| **Relatedness** (关联感) | 最小化——纯单人精通体验，无社交需求 | Minimal |

### Player Type Appeal (Bartle Taxonomy)

- [x] **Achievers** — 无伤通关、完美连击、计分排行榜是成就追求
- [x] **Explorers** — 发现招式组合和隐藏连招是探索动力
- [ ] **Socializers** — 不适用（纯单人）
- [ ] **Killers/Competitors** — 次要吸引力（计分系统可扩展为排行榜）

### Flow State Design

- **Onboarding curve**: 第一波敌人只用一式即可应对，逐步引入第二式、第三式，最后要求三式组合
- **Difficulty scaling**: 波次难度递增 + 敌人种类多样化，迫使玩家使用更多剑式
- **Feedback clarity**: 流光轨迹、命中音效、连击计数器三重反馈，玩家随时知道自己的表现
- **Recovery from failure**: 瞬间重启，无加载惩罚。死亡即学习，失败不惩罚

## Core Loop

### Moment-to-Moment (30 seconds)
观察敌人 → 判断用哪一式 → 精确出招 → 墨色流光或金色剑气反馈 → 视觉/听觉确认命中。核心满足感来自精确命中时材质碰撞的真实反馈——火花、裂纹、墨点炸碎——加上轨迹完美贴合剑尖的视觉确认。

### Short-Term (5-15 minutes)
竞技场波次战——敌人一波接一波，每波引入新敌人类型。三式组合使用，连击积累走向万剑归宗。每一波都是一个小型挑战，"再来一波"的心理驱动。

### Session-Level (15-60 minutes)
完整一局：从第一波到最后的极限波次。难度持续递增直到死亡。每次运行产生一个新的"最佳记录"——最远波次、最长连击、最快万剑归宗。自然停止点是死亡后的喘息，但"再来一次"的冲动很强。

### Long-Term Progression
**纯技巧进步**——没有属性升级，没有解锁新招（或仅解锁变体，不提升数值）。玩家的进步体现在：更长的连击、更准的出招、更快的反应、更频繁的万剑归宗。游戏的"完成"不是通关，是玩家自己觉得"我掌握了"。

### Retention Hooks
- **Mastery**: 每次运行都是操作练习，手感提升是即时可感知的
- **Curiosity**: 三式组合的变体和隐藏连招驱动探索
- **Investment**: 最佳记录和计分系统让每次运行都有意义
- **Social**: 无（单人体验）

## Game Pillars

### Pillar 1: 精确即力量 (Precision is Power)
每一剑的角度和时机都有意义。没有无脑连按，只有精准出招。

*Design test*: 当犹豫"加更多特效还是保持清晰"时，选清晰。

### Pillar 2: 极简即美学 (Minimalism as Aesthetics)
流光追踪剑尖，不加多余粒子。少即是多，每一帧都有意义。

*Design test*: 当犹豫"加一个效果会不会更酷"时，问"去掉它会失去什么"——如果不会，就不加。

### Pillar 3: 玩家即进度 (Player is Progression)
没有属性升级。所有成长来自你的手法进步——更长的连击、更准的出招、更快的反应。

*Design test*: 当犹豫"加永久升级还是纯技巧"时，选纯技巧。

### Pillar 4: 三式皆平等 (Three Forms, Equal Weight)
游剑式、钻剑式、绕剑式各有不可替代的价值。没有"最强形态"。

*Design test*: 如果一个形态在 80% 场景都最优，说明它需要被削弱。

### Pillar 5: 万剑归宗即高潮 (Climax Through Mastery)
连击触发的万剑归宗是玩家技巧的最高表达。它的壮观程度和你投入的精确度成正比。

*Design test*: 万剑归宗不应该靠运气触发，只应该靠技巧触发。

### Anti-Pillars (What This Game Is NOT)

- **不做数值成长**: 会腐蚀"玩家即进度"支柱——如果玩家可以通过刷属性通关，精确就不再重要
- **不做华丽特效**: 会腐蚀"极简即美学"支柱——粒子雨、镜头光晕会掩盖剑招本身。万剑归宗用密度代替华丽
- **不做打断节奏的叙事**: 允许环境叙事和碎片化叙事，但不允许强制过场打断战斗节奏
- **不做多人**: 这是一款纯粹的个人精通体验——竞技性会把焦点从自我提升转移到赢别人

## Visual Identity Anchor

**方向名称**: 山水 (Shanshui)

**视觉法则**: 水墨山水为骨架，极简几何为血肉。场景如流动的水墨画。浩然剑气无需表现在剑锋之上。

**色彩哲学**: 灰白水墨为底。墨色流光与金色剑气交替穿梭。留白即呼吸空间。

**设计测试**: 如果一个元素不能像一笔墨一样简洁有力，就重做它。

### 三式视觉与交互特征

#### 绕剑式 —— 以柔克刚
- **视觉**: 墨色流光绕着身体旋转，如流动的水墨太极
- **交互**: 碰到敌方攻击时墨点炸碎，"以柔克刚"——用柔化解刚
- **色彩**: 墨黑流光，碎裂时溅射为水墨粒子
- **感觉**: 防御性、化解性、太极哲学

#### 游剑式 —— 以巧制敌
- **视觉**: 金色剑气像小蛇缠绕敌刃，灵动而精确
- **交互**: 碰到金属刀兵冒火花，碰到木杖裂细纹——材质反应式交互，真实感拉满
- **色彩**: 金色流光，材质碰撞产生对应色火花/裂纹
- **感觉**: 灵活性、附着性、四两拨千斤

#### 钻剑式 —— 以力破防
- **视觉**: 剑尖凝聚高密度金色粒子，形成锐利的光锥
- **交互**: 刺穿防御时炸开扇形冲击波，连城墙都裂出深沟——破坏力肉眼可见
- **色彩**: 高密度金色粒子聚爆，冲击波为扩散的金白色
- **感觉**: 穿透性、破坏性、一击定乾坤

### 三式对比总结

| 特征 | 绕剑式 | 游剑式 | 钻剑式 |
|------|--------|--------|--------|
| **色彩** | 墨黑 | 金色 | 金色高密度 |
| **角色** | 防御/化解 | 机动/附着 | 穿透/破坏 |
| **哲学** | 以柔克刚 | 以巧制敌 | 以力破防 |
| **轨迹** | 环绕身体 | 缠绕敌刃 | 凝聚剑尖 |

## Technical Considerations

| Consideration | Assessment |
| ---- | ---- |
| **Recommended Engine** | Godot 4.6.2 stable (win64) — 用户已有 Godot 经验，Web 导出支持良好 |

## Risks and Open Questions

### Design Risks
- **纯技巧无数值成长是否能维持"再来一局"动力？** — 需要通过原型验证。如果连击和万剑归宗的满足感足够强，不需要数值成长作为诱饵。
- **三式平衡是否可行？** — 需要大量测试确保没有"最强形态"，三式各有不可替代的使用场景。

### Technical Risks
- **Godot 4.6.2 HTML5 3D 性能** — 高风险。Web 平台的 3D 渲染有严格性能限制，特别是动作游戏的 60fps 要求。需要在原型阶段验证。
- **实时精确碰撞检测** — 中风险。精确打击需要精准 hitbox，Web 环境下可能有延迟。
- **低多边形水墨着色器** — 中风险。需要自定义着色器实现水墨风，WebGL 兼容性需验证。

### Market Risks
- **Web 平台的 3D 动作游戏是否有人玩？** — Web 游戏以 2D/休闲为主，3D 动作在 Web 上是小众领域。但极简风格降低了门槛。
- **中国剑侠题材在国际市场接受度** — 需要考虑国际化元素。

### Scope Risks
- **完整 Demo 范围对个人开发者可能偏大** — 建议先完成 MVP（Tier 1），再扩展到完整 Demo（Tier 2）。
- **自定义着色器的工作量被低估** — 水墨风格着色器可能需要大量调试。
- **3D 资产制作是最大瓶颈** — 5种敌人 + 玩家角色 = 6个角色的建模/动画。需提前确定资产来源策略（自建/购买/简单几何体占位）。
- **招式变体系统复杂度膨胀** — "每式2个变体"可能从3式变成6式的工作量。

### Open Questions
- Godot 4.6.2 的 HTML5 导出对 3D 游戏的性能上限是多少？→ 在原型阶段用 Benchmark 测试
- 低多边形水墨着色器在 WebGL 下的效果如何？→ 制作着色器原型验证
- 游戏导出为 ZIP 包后的文件大小预算？→ 目标 50MB 以内
- 3D 资产来源策略？→ 自建 / 购买素材包 / 简单几何体占位

## MVP Definition

**Core hypothesis**: "精确打击"的手感在 3D 竞技场中是否成立——三式切换、流光轨迹、连击触发万剑归宗，这些是否能让玩家在 5 分钟内就产生"再来一次"的冲动。

**Required for MVP**:
1. 一个竞技场场景（山水风格低多边形）
2. 一种基础敌人（可被三式击败）
3. 三式基础剑招（游剑式、钻剑式、绕剑式）+ 流光轨迹
4. 万剑归宗连击触发

**Explicitly NOT in MVP**:
- 多区域/多敌人类型
- 招式变体系统
- 计分系统
- 完整 UI（菜单、HUD）

### Scope Tiers

| Tier | Content | Features | Timeline |
| ---- | ---- | ---- | ---- |
| **MVP** | 1个竞技场、1种敌人 | 三式基础招、流光轨迹、万剑归宗触发 | 核心验证 |
| **完整 Demo** | 2个区域、5种敌人 | 每式2变体、计分系统、关卡选择 | 你的目标 |
| **扩展愿景** | 3+区域、Boss战 | 完整招式树、排行榜、回放系统 | 未来扩展 |
