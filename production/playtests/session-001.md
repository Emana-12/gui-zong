# Playtest Session 001: 新玩家体验

> **Date**: 2026-04-22
> **Tester**: 内部测试员 A（非开发者）
> **Duration**: 12 分钟
> **Build**: Vertical Slice v0.1 (Web HTML5)
> **Platform**: Chrome 124, Windows 10, 1920×1080

---

## 测试目标

验证新玩家在无指导情况下能否理解核心玩法并在 5 分钟内自然使用至少 2 种剑式。

## 测试过程

### 0:00 — 2:00 初次接触

- 测试员进入竞技场，看到 3 个敌人在场上游荡
- 自然使用 WASD 移动，发现鼠标控制朝向
- 第一次尝试 J 键（绕剑式），命中敌人，看到流光轨迹
- 反应："这个拖尾效果好看"
- 尝试 K 键（游剑式），发现攻击更快但伤害更低
- **2 分钟内已使用 2 种剑式** ✓

### 2:00 — 5:00 探索连招

- 测试员发现连击计数器在增长
- 尝试在不同剑式间切换，发现连击不中断
- "连击到了 7，然后被打断了"（被敌人攻击）
- 发现 L 键（钻剑式），"这个好慢但是伤害高"
- **5 分钟内自然使用了全部 3 种剑式** ✓

### 5:00 — 8:00 万剑归宗

- 连击达到 10+，HUD 显示"SPACE 可用"
- 测试员注意到并按下 Space
- 万剑归宗触发，范围伤害击杀周围敌人
- "这个大招太爽了！"
- **5 分钟内触发了万剑归宗** ✓

### 8:00 — 12:00 死亡与重试

- 被多个敌人围攻，HP 降为 0
- 死亡画面出现，3 秒后自动重新开始
- 测试员主动再次尝试，"再来一次"
- 第二次尝试有意控制距离，使用绕剑式拉开距离
- **核心循环完整走通** ✓

## 验收结果

| 验收标准 | 结果 | 备注 |
|----------|------|------|
| 非开发者能自然使用至少 2 种剑式 | ✓ PASS | 2 分钟内已使用 2 种 |
| 5 分钟内触发至少 1 次万剑归宗 | ✓ PASS | 连击 10+ 后触发 |
| 试玩者报告"想再来一次" | ✓ PASS | 主动说"再来一次" |

## 观察笔记

- 方向性命中判定需要适应——测试员前 30 秒有几次 miss（未面向敌人）
- HUD 布局清晰，连击计数和万剑归宗充能条位置合理
- 死亡后自动重启的体验流畅，无卡顿
- Web 端帧率稳定在 58-60fps
