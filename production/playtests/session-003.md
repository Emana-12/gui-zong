# Playtest Session 003: 难度曲线与核心循环耐玩性

> **Date**: 2026-04-22
> **Tester**: 内部测试员 C（中等动作游戏经验）
> **Duration**: 25 分钟
> **Build**: Vertical Slice v0.1 (Web HTML5)
> **Platform**: Firefox 125, Windows 11, 1920×1080

---

## 测试目标

验证长时间游玩体验：难度是否合理、是否存在"困惑循环"（玩家卡住不知道原因）、核心循环是否耐玩。

## 测试过程

### 第一轮（0:00 — 8:00）

- 初次上手，花 1 分钟理解操作
- 第 3 分钟首次击杀敌人，建立信心
- 第 5 分钟首次触发万剑归宗，击杀 3 个敌人
- 第 7 分钟被围攻死亡，得分 45
- 评价："节奏刚好，不会太难也不会太无聊"

### 第二轮（8:00 — 15:00）

- 熟练使用三式切换，连击效率明显提升
- 有意使用钻剑式打高伤害，游剑式攒连击
- 第 12 分钟触发第二次万剑归宗
- 第 14 分钟死亡，得分 78（比第一轮高 73%）
- 评价："知道怎么玩了，想打破记录"

### 第三轮（15:00 — 22:00）

- 开始主动控制距离，利用敌人 RETREAT 间隙输出
- 使用绕剑式的流光轨迹判断攻击范围
- 连击最高到 16，万剑归宗伤害=21
- 第 20 分钟死亡，得分 112
- 评价："越玩越好，有成长感"

### 冷却期（22:00 — 25:00）

- 测试员主动继续第四轮
- 没有表现出厌倦
- 评价："如果有多几种敌人类型会更好"

## 困惑循环检测

| 检查项 | 结果 |
|--------|------|
| 是否有玩家卡住不知道原因的时刻？ | 否 |
| 是否有操作后结果不符合预期的时刻？ | 否 |
| 死亡原因是否可理解？ | 是——每次都知道被哪个敌人打的 |
| 重新开始后是否知道该做什么？ | 是——自然进入战斗 |

**结论：无困惑循环** ✓

## 难度曲线评估

| 阶段 | 玩家体验 | 难度评价 |
|------|----------|----------|
| 初次接触（0-3 分钟） | 学习操作，低压力 | 适中 |
| 建立信心（3-8 分钟） | 击杀敌人，触发大招 | 略低（可接受——垂直切片） |
| 熟练期（8-15 分钟） | 连击效率提升 | 合理 |
| 挑战期（15-22 分钟） | 追求高分 | 有挑战 |

**当前难度适合垂直切片**。正式版需要更多敌人类型和波次递增。

## 耐玩性评估

| 指标 | 结果 |
|------|------|
| 是否想再来一轮？ | 是——测试员主动玩了 4 轮 |
| 是否感到厌倦？ | 否——25 分钟内未感到厌倦 |
| 核心循环是否有深度？ | 是——三式切换+距离控制提供决策空间 |
| 是否有成长感？ | 是——得分从 45 → 78 → 112 逐轮提升 |

## Verdict: PASS

难度曲线合理，无困惑循环，核心循环耐玩，玩家有成长感。
垂直切片验证通过——正式版需增加敌人种类和波次系统以维持长期吸引力。
